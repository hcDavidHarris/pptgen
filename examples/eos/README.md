# EOS Example Library for pptgen

This folder contains ready-to-copy YAML examples for common EOS artifacts.

Included examples:
- eos_scorecard.yaml
- eos_rocks.yaml
- eos_issues_list.yaml
- eos_vto.yaml
- eos_quarterly_review.yaml

Suggested repo location:

```text
pptgen/
└─ examples/
   └─ eos/
```
